###### **/auxShare/**

this folder is network-shared across the B-link, SOS2 server, and Raspberry Pi 5



**auxShare/audio/**

this folder contains all related files to automated audio handling



**auxShare/audio/audio-config.JSON**

this file contains audio file names organized by category, modified only by facilitator commands found in engine.py



**auxShare/audio/audio-list.CSV**

this file contains a list of audio file names and their associated category in key:value format



**auxShare/audio/mp3**

this folder contains all the .mp3 files for playing ambient and narrated audio



###### **auxShare/cache**

Contains runtime cache files generated by cache_manager.py to minimize SOS server queries

**auxShare/cache/subtitles**

Contains cached .srt subtitle files fetched from the SOS server for offline subtitle rendering

**auxShare/cache/clip\_metadata\_cache.JSON**

Contains clip metadata (duration, captions, titles, categories) generated by cache\_manager.py from SOS server queries

**auxShare/cache/playlist\_cache.JSON**

Contains playlist structure (clip names and order) generated by cache\_manager.py from the SOS server

###### **auxShare/data**

Contains SOS dataset information for PowerPoint slide navigation, initialized by sdc.py, pp\_init.py, and pp\_access.py

**auxShare/data/SOS_datasets.csv**

Maps dataset names to English/Spanish titles and slide numbers for LibreOffice navigation

###### **auxShare/documents**

Contains the PowerPoint/LibreOffice presentation displayed on ring monitors, managed by pp\_access.py

**auxShare/documents/noaa_presentation.odp**

LibreOffice Impress presentation file with slides synchronized to SOS datasets

###### **auxShare/assets**

Contains shared font files used by PyQt5 overlay components

**auxShare/assets/Inter_18pt-Medium.ttf**

Font file for standard subtitle and progress bar text rendering

**auxShare/assets/Inter_18pt-Bold.ttf**

Bold font file for emphasized subtitle text rendering


###### **/main/**

**main/sdc.py**

- Main entry point that initializes CacheManager, LibreOffice, audio system, and launches the engine
- Connects to SOS server to retrieve current playlist and synchronizes cache files
- Sends initial playlist data to nowPlaying display on Raspberry Pi

**main/cache_manager.py**

- Manages all caching operations to minimize SOS server queries during runtime
- Handles playlist and clip metadata caching with sync() based on modification dates
- Fetches and caches subtitle files locally for offline rendering
- Manages audio configuration and category tracking for ambient audio playback

**main/pp_init.py**

- Initializes LibreOffice Impress with the .odp presentation file
- Loads slide mapping dictionary from CSV database
- Creates slide dictionary mapping dataset names to slide numbers for navigation

**main/pp_access.py**

- Provides PowerPointShowController class for LibreOffice Impress control via Windows COM
- Handles slide navigation, presentation lifecycle, and slideshow management
- Includes automatic crash recovery and connection handling

**main/audio_init.py**

- Loads audio file mappings from CSV file organized by category
- Initializes AudioController for MPV-based audio playback on SOS2 server
- Returns audio dictionary and controller instance for use by engine

**main/audio_access.py**

- Provides AudioController class for remote MPV audio playback via SSH and IPC
- Manages audio lifecycle including play, pause, fade in/out, and volume control
- Handles SSH connection, socket communication, and playback state tracking

**main/engine.py**

- Core engine that monitors SOS server for clip changes by probing clip_number every 50ms
- Manages LibreOffice slide navigation, subtitle display, and progress bar overlays
- Controls ambient audio playback based on dataset major categories
- Handles overlay visibility logic for credits, translated movies, and standard datasets
- Runs query server for nowPlaying to request current state information

**main/overlay_subtitles.py**

- Parses .srt subtitle files and synchronizes subtitle display with playback time
- Provides SubtitleOverlay PyQt5 widget for dual-language subtitle rendering
- Manages subtitle visibility, positioning, and fade animations

**main/overlay_progressBar.py**

- Provides ProgressBarOverlay PyQt5 widget for playback progress visualization
- Displays custom progress bar
- Shows current time and total duration 
- Handles fade in/out animations and positioning
