1_sos-test_connect.py
Connects to the SOS server and returns the metadata of the currently playing clip. 

2_sos-index_generator.py
Creates an index of all SOS dataset names via SSH, extracts clip names from metadata and organizes them into a CSV file, saved locally. Takes a long time.

3_sos-query_metadata.py
Prompts for a metadata type, then retrieves that metadata value for all clips in the currently playing playlist. Returns a list of clip names with their corresponding queried metadata values. 

4_sos-playlist_all_metadata.py
Retrieves complete metadata for all clips in the current playlist. 

5_sos-download_playlist_file.py
Displays all clips in the currently playing playlist, allows user to download associated playlist.sos dataset files (containing metadata) locally from the SOS server.
