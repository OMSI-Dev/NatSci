(Issue: ) Connecting a new audio device for audio output to the SOS system

(Solution: ) Re-configure the device with pipewire

(Step 1: ) Identify the mixer
	aplay -l
	wpctl status

(Step 2: ) Set it as default
	wpctl set-default <device-id> 

(Step 3: ) Test
	paplay /usr/share/sounds/alsa/Front_Center.wav

(Step 4: ) Apply changes over reboots
	mkdir -p ~/.config/wireplumber/main.lua.d/
	nano ~/.config/wireplumber/main.ua.d/51-default-sink.lua

	(4a: ) Add:
		default_sink_name = "alsa_outut.<your-mixer-device-string>"
